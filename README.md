\# 노션 개인 홈페이지 템플릿



PICTORY 개인 포트폴리오 홈페이지를 기반으로 만든 노션 템플릿입니다.

노션 마켓플레이스에서 무료 복제하여 바로 사용할 수 있습니다.



\## 템플릿 구성



\- 메인 홈페이지

\- Profile (About)

\- Portfolio (Works)

\- Contact



\## \_archive 폴더



템플릿 제작 과정에서 사용한 보조 스크립트 모음입니다.



| 파일 | 설명 |

|------|------|

| `add\_portfolio\_guide.js` | 포트폴리오 가이드 추가 |

| `check\_final\_portfolio.js` | 최종 포트폴리오 검증 |

| `get\_portfolio\_blocks.js` | 포트폴리오 블록 조회 |

| `update\_contact.js` | 연락처 페이지 업데이트 |
